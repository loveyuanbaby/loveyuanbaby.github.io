写在春招实习面试结束时

发表于 2018-04-18 | 分类于 diary | | 阅读次数

经历了三月的努力学习，积极准备，虽然投出去的实习简历比较少，面试机会也不多，但还算比较顺利地经过了四轮面试拿到了头条的实习offer，个人感觉还是挺满意的，但是，满意有什么用呢？老板不放人啊。

抨击下当前中国的研究生导师制，导师权力过大，掌握了研究生毕业的生杀大权，当一个教授不再把自己看作是一个教授而是一个老板，不再把研究生看做需要自己指导学术的学生，而是自己的廉价劳动力，这是多么可怕又多么可悲的一件事。学生无力反抗，为了毕业只能成为任人宰割的羔羊。最近有很多研究生不忍压迫用生命抗争的新闻，唏嘘不已的同时，我们研究生也都知道其实这些不是个案，只要制度不改这些事情不会是悲剧的开始更不会是悲剧的结束。

很不幸，我就是这样一位廉价劳动力，在一个还算过得去的985院校读研，从入学开始就在导师的公司打工。公司距离学校很远，每天要浪费三个多小时的时间在路上，为了及时赶到公司必须要早起，有多早呢？6点半。然后一看一天的电脑，每天如此整个人都憔悴了。长此以往，似火的学术热情也消磨殆尽。

额，竟然写了这么多了，有些话还是等到毕业再写吧。

既然不能出去实习，远宝宝我也不会停止前进的脚步的。我打算把周报里面比较有营养，自己做的比较成功的东西放到博客里，一来感谢其他写博客的大神的文章的帮助，二来给自己个安慰，我远宝宝还是学了很多，做了很多的。
